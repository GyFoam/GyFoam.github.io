INTELLITEX

Pinball game input

Anonymous GitHub repo

Taking advantage of the flexibility and elasticity of textiles, we combined our approach with concepts of textile sculpture and fabricated an interface using highly elastic fabrics. Four layers of elastic fabric are combined to sense a wider range of push inputs.

Design files (laser-cut files) can be find in the github repo. Fabric pieces were connected to the acrylic base using conductive silver thread. Diagonal resistance of each fabric layer was connected to the processor component to determine the extent of push input.

Through this example, we showcased that the IntelliTex input does not necessarily have to be implemented on textiles. Instead, it can be combined with other materials, such as acrylic. We further demonstrated its interaction potential by using it as an input for a pinball game, serving as a realistic force simulator for pressing the spring in the virtual game.
